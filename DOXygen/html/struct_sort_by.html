Detailed Description

The purpose of this struct is to be used as the sorting predicate for the multiset TheBeers.

Date
Created on April 07, 2015

The SortBy struct contains a static variable that may be used in determining how the user wishes to sort the list. By default we sort by kName (the variable is initialized just above the main function). But, if the user wishes to re-sort a they access this through the appropriate menu option
