Detailed Description

The purpose of this struct is to be used as the search predicate for the find_if function used to search the BeerFridge.

Date
Created on April 04, 2015

The FindBeer struct contains four variables that may be used in the user search of the database. When find_if calls FindBeer it passes to it the string that is being searched for, the alcohol content or the IBU count. Lastly, and most importantly is the 'search by' which is used to determine which case in the overloaded () will be used.
