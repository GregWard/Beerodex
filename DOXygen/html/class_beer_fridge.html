Detailed Description

The purpose of this class is to instantiate a new BeerFridge object to contain the multiset of beers, as well as methods to access and change the contents/how they are displayed.

Date
Created on March 30, 2015

The BeerFridge class contains the typdefed Multiset TheBeers
